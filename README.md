## Features

- Create a user profile
- Log workouts with exercise name, weight, and reps
- Save workout history permanently using file handling
- View complete workout history from within the application
- Automatic date and time stamping for each workout entry
- Personal Record (PR) tracking for any exercise
- Menu-driven terminal interface
- Object-Oriented Programming (OOP) structure using classes
- Modular code organization across multiple Python files

## Concepts Used

This project helped me practice:

- Classes and Objects
- Functions
- Loops and Conditionals
- File Handling
- String Formatting (f-strings)
- Python Modules and Imports
- Datetime Library
- Basic Data Processing
- Debugging and Problem Solving
