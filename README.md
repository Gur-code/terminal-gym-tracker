# Terminal Gym Tracker

A terminal-based gym tracking application built using Python.

Features:
- Log workouts
- Save workout history
- Timestamp logging
- Persistent file storage
- Menu-driven interface

Concepts Used:
- OOP
- File Handling
- datetime
- Loops
- Functions
- Modular Programming

Run:
python main.py
